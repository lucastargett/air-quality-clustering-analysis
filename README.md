# Air Quality Clustering Analysis

Main notebook: [air_quality_clustering_analysis.ipynb](./air_quality_clustering_analysis.ipynb)

Machine learning clustering and dimensionality reduction analysis using KMeans, DBSCAN, PCA, Hierarchical Clustering, and OPTICS on environmental sensor data.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Techniques Applied
- KMeans Clustering
- DBSCAN
- Hierarchical Clustering
- OPTICS
- PCA
- Silhouette Score Evaluation
- Data Preprocessing & Visualisation
